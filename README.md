# Deep_learning_WUR
Deep Learning Project Assignment

Nivel 1: SmallCNN from scratch. Evaluate the groundbase results.

Level 2: Pre-trained ResNet-50 in ImageNet with fine-tuning. It's the current best options considering time and resources limitations.

Level 3 (Innovation part): Pre-trained ViT-B/16. CNNs don't capture global dependencies from the first layer, while ViTs do. We'll measure whether that difference translates into better accuracy in the workflow. It might, it might not—both conclusions are valid and reportable.
